# Hive - 沉浸式在线自律陪伴 App 🐝

## 一、 项目概述
- **产品定位**：面向苹果 (iOS) 及移动环境的、主打“高级感、克制、沉浸”的虚拟自律自习室。
- **核心 Slogan**：同处蜂巢，各自闪能。 (Together in the Hive, Shining Alone.)
- **设计美学**：极简与克制（Less is More），采用极具沉浸感的深度暗黑/钛灰色背景，辅以“琥珀金”与“智能光晕”点缀，并深度利用 iOS 风格的毛玻璃模糊（Background Blur）特效。

## 二、 目标用户
- 需要**深度专注**（如考研、深度办公、阅读、编程等）的用户。
- 渴望陪伴感，但**极度排斥无效社交**与文字聊天打扰的群体。
- 追求高质感 UI/UX 以及丝滑交互体验的颜控及效率控用户。

## 三、 核心功能说明

目前的本地代码库（尤其是 `App.tsx`）已经实现了以下核心模块：

### 1. The Hive (蜂格空间 / 自习室大厅)
- **空间视觉呈现**：放弃传统头像列表，采用 **HiveGrid（六边形网格）**的排布。当前在线同伴以呼吸灯的光效展现在六边形蜂巢中。
- **多维度视图**：
  - **Squad（小队）**：展示具体绑定在一起自律的极少数熟人/搭子。
  - **Global / Region (全息星球视窗)**：可以通过切入全屏 3D 地球模式（基于 WebGL 渲染），查看全球其他地区的蜂格点亮情况。

### 2. Digital Bonds (数字羁绊与极简社交)
- **无言的陪伴**：互为 Bond 的用户在专注期间**禁止私聊**，只能看到对方当前在学习的科目（Subject）与时空位置。
- **Nudge（实时轻推）**：如果看到好友正在专注，可以通过点击他们的蜂格发送一个“轻推”。系统会通过 WebSocket 触发对方手机的**物理马达震动 (Haptic Feedback)**，并在屏幕上瞬间闪现一道夺目的光效作为鼓励。

### 3. Focus Engine (沉浸式专注核心)
- **长按仪式感**：采用长按积累光圈的机制进入专注，增强仪轨感。
- **环境白噪音与特效**：进入专注后屏幕浮动环境光标（Ambient Particles），界面中心只留下极致清晰的倒计时环和当前状态提示。
- **Sparks（星火奖励）**：每一次完成专注后，可以获得 `Sparks` 奖励，用于评估总自律投入度。

### 4. Subscription & Management (会员与试用机制)
- 内置 Trial (7天试用期) 计算机制，带有极简紧凑的订阅到期提示面板和相关管理链路。

## 四、 技术架构选型

项目采用了一套**基于 Web 技术构建却极具原生性能体验**的跨端全栈架构：

### 1. 前端模块 (Frontend)
- **React 19 + TypeScript**：核心视图层与逻辑骨架。
- **Vite**：极速的热更新与打包。
- **Capacitor**：将 Web 应用编译包装为 iOS/Android 原生 App，并调用底层硬件级能力（例如：震动 Haptics、IAP 内购、Apple 登录）。
- **Tailwind CSS**：完成高级暗黑深灰基调、像素级别的样式还原与响应式排版。
- **Three.js**：专门渲染宏观大地图模式下的 3D 地球。
- **i18next**：提供多语言国际化支持。

### 2. 后端模块 (Backend)
- **Python + FastAPI**：具有极高并发处理能力且原生支持 WebSocket 的高性能框架，支撑毫秒级的轻推和在线状态广播。
- **SQLAlchemy (asyncio) + SQLite/PostgreSQL**：异步持久化存储用户关系、记录和小队。
- **Pydantic / Passlib / Python-jose**：涵盖 JWT 鉴权、数据结构验证及密码学加解密的核心组件。

## 五、 产品路线与演进方 (Roadmap)

1. **已达成里程碑 (MVP已跑通)**：
   前后端关键链路已连通，涵盖 Auth 登录、加入 Squad / 添加 Bond、长按开启倒计时、WebSockets 实时共享专注情况以及专注结算逻辑。
2. **重点演进方向**：
   - **Focus Lock (防切出机制)**：依托 iOS Screen Time API 或插件阻断用户跳出，打造不分心的体验。
   - **Hex-Stats (成就可视化)**：将 Sparks 历程转为全息的 3D 数据面板及六边形画卷。
   - **IAP 内购闭环与经济机制**：深度接入 Apple App Store 收银台以完善付费试用/会员转化链路。
